How to Use the Shift Report

  1. Upload Sales: Select your "Line Transaction" CSV file. This contains all itemized sales and voids.
  2. Upload Payments: Select your "Payments" CSV file. This tracks actual money collected.
  3. Filter by Staff: Once both files are uploaded, use the Staff dropdown to isolate one person's performance.
  4. Analyze Data:
    • Sales Table: View "Own Sales" vs "Made by Others" for every account.
    • Payments Table: See exact payments collected by that staff member.
  5. Shaded Totals: The dark-shaded rows represent the calculated Grand Totals for each section.
